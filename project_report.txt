PROJECT REPORT: AGENTFORGE AI HIRING AGENT

Overview:
AgentForge is an AI-powered, multi-agent recruitment automation system designed to streamline the full hiring lifecycle for technical roles.

AgentForge is a multi-agent AI orchestrated recruitment pipeline that humanizes automation. It acts as an autonomous HR department, handling everything from initial intake to final verdict:

1. Resume Screening – AI evaluates candidate fit for MERN Stack roles.
2. Background Verification – Validates email, LinkedIn, and GitHub profiles.
3. Interview Question Generation – Tailored technical questions from the resume.
4. Voice Interview – Vapi AI conducts a live phone interview.
5. Evaluation – Transcript + resume scored and assessed.
6. Final Decision – HR notified via Slack; candidate data stored.

AgentForge is built on a modern, robust, and highly scalable architecture:
- LLM: Llama 3 (8B Instruct) via OpenRouter API
- Voice: Vapi.ai for outbound telephony and transcription
- Backend: FastAPI (async) + Streamlit (prototype)
- Frontend: Next.js + TypeScript
- Storage: JSON file-based candidate database
- Notifications: Slack Incoming Webhooks
- Email: Gmail SMTP

Implementing the AgentForge AI Hiring Agent transforms the hiring lifecycle:
- Reduces recruiter admin time by 70%+
- Provides instant candidate screening and verification
- Conducts AI-driven voice interviews 24/7
- Delivers clear, structured hiring decisions to HR

AgentForge is not just an automation tool; it is a force multiplier for modern HR teams. By offloading the administrative and repetitive tasks of technical recruitment to a specialized AI agent, companies can focus on what truly matters: building great teams with the right talent.
